# Valpo-Ignis
Proyecto de aplicacion GPS enfocada en enviar alertas de zonas de incendio, la cuál con input de usuario indicar una zona radial donde ocurre un incendio, la intensidad del mismo y un calculo por aplicación para encontrar rutas habiles de escapes con hotline para servicios de bomberos. 

Integrantes: 
Nicolas Lazcano,  Sebastian Rojas, Angelo De La Rivera y Rodrigo Sances
